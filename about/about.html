About

This website documents the collective work of Group Two on the project "Everything But the Film Musical" for the class MUSC 80H (Winter Quarter 2021) at UCSC. We have adapted the 2000 animation Chicken Run into a hypothetical musical film that is set in America during the industrial revolution.

Disclaimer

This website only contains the work that was made available in the shared Google Drive for Group Two. It is possible that other members of the group completed work for the project individually that is not documented on this site.
